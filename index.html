Este é um exemplo de teleprompter. O texto rola automaticamente para que você possa lê-lo sem precisar rolar manualmente. Com os botões você consegue navegar e configurar seu teleprompter. Você consegue abrir um documento do Google Docs (desde que esteja público o acesso) e utilizá-lo aqui.

Google Docs Link

Lembre-se seu documento precisa estar visível como "Qualquer pessoa com link"

AJUDA

Você também pode controlar seu teleprompter pelo teclado
ESPAÇO - Inicia/Pausa rolagem
A - Aumenta Velocidade
S - Diminui Velocidade
D - Alterna Dark/Light mode
F - Espelha tela (Flip)
Q - Aumenta fonte
W - Diminui fonte
